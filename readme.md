# timestamp2date

一个用于将时间戳转换为日期格式的工具。

## 命令行使用

npm i timestamp2date -g

```bash
timestamp2date
```

或

```bash
t2d
```

## es6 模块使用

```bash
npm i timestamp2date
```

```javascript
import timestamp2date from 'timestamp2date';
console.log(timestamp2date(1678297600000));
```

## commonjs 模块使用

```bash
npm i timestamp2date
```

```javascript
const timestamp2date = require('timestamp2date');
console.log(timestamp2date(1678297600000));
```
